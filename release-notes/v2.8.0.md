# Restate Operator v2.8.0 Release Notes

**v2.8.0 was never published — it is superseded by [v2.8.1](https://github.com/restatedev/restate-operator/blob/main/release-notes/v2.8.1.md).**

v2.8.1 ships CRDs as an optional `restate-operator-crds` subchart using Helm's native `crds/` directory instead, gated by `installCrds` (default `true`). See the [v2.8.1 release notes](https://github.com/restatedev/restate-operator/blob/main/release-notes/v2.8.1.md).
